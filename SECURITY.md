# Security Policy

## Supported Versions
We actively maintain and support the following versions of my-prose-App:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | :white_check_mark: Supported |
| < 1.0   | :x: Unsupported |

## Reporting a Vulnerability
If you discover a potential security vulnerability in this project, please **do not** open a public issue. Instead, follow these steps:

1. Send a detailed report describing the vulnerability to the project maintainers.
2. Include a clear description, step-by-step instructions to reproduce it, and any potential impact.

We will acknowledge receipt of your vulnerability report within 48 hours and work on a resolution as quickly as possible.
